HISTÓRIA DOS COMPUTADORES

VOLTAR

Chamado popularmente de Mark I, foi a primeira calculadora eletro-mecânica automática desenvolvido nos Estados Unidos, projetada em 1930 pelo estudante pós-graduando em física Howard Aiken, na Universidade Harvard e, construído em 1944, em uma parceria com a empresa IBM, durante a Segunda Guerra Mundial.

A Calculadora Automática de Sequência Controlada (Harvard Mark I) foi a primeira máquina operacional que poderia executar cálculos longos automaticamente.

Pesando cerca de 5 toneladas foi a primeira calculadora automática produzida em larga escala.

O QUE SÃO CLIENTS

VOLTAR

O client é quando você consome a informação de algum lugar, baseado no nosso mundo web, o client é o dispositivo que está consumindo aquela informação da internet.

Eles se comunicam através da internet e por outros computadores. A interação entre os processos client e servidor é uma troca cooperativa na qual o client é ativo e o servidor reativo, ou seja, o client requisita uma operação e, nesse ponto, o servidor processa e responde ao cliente.

Quando você faz um acesso no site, ele te retorna arquivos HTML, arquivos CSS, arquivos JAVA SCRIPT, arquivos de ÁUDIO/IMAGEM. Estes arquivos são chamados de arquivos estáticos, são enviados do servidor para o Client. E isso só é possível graças a internet do usuário que é o que faz conexão entre o Client e o Servidor.

O QUE SÃO SERVERS

VOLTAR

O servidor usa conexão de internet para estabelecer uma ligação com o Client. Porém, isso não quer dizer que o servidor pode se conectar com o usuário somente pela internet. Ele pode ser um servidor que se conecta por uma Rede Lan (Local Area Network).

Muitas empresas utilizam esse sistema de Lan, onde tem um computador central que armazena alguns documentos/arquivos. Aquela máquina é uma rede local, onde outras máquinas da mesma rede conseguem acesso a ela por isso se torna oficialmente um Servidor.

Na maioria das vezes os Servidores onlines são feitos com o intuito principalmente de segurança. Para armazenar aplicações, relatórios, documentos, planilhas e rotinas administrativas da empresa.

DIFERENÇA ENTRE LISTA ORDENADA E NÃO ORDENADA

VOLTAR

    O que é uma lista ordenada?

  1. Uma lista ordenada é também uma lista de itens. As listas de itens são marcadas com números.
  2. Uma lista ordenada começa com a tag "ol";
  3. Cada item da lista começa com a tag "li".

    O que é uma lista não ordenada?

  • Uma lista não ordenada é uma lista de itens. As listas de itens são marcadas com bullets (tipicamente pequenos círculos pretos).
  • Uma lista não ordenada começa com a tag "ul";
  • Cada item da lista começa com a tag "li".
